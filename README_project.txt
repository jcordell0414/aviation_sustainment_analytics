
# Aviation Maintenance & Sustainment Analytics Project

## Objective
Analyze aircraft maintenance operations, downtime, supplier delays, and readiness performance 
using SQL, Python, and Tableau.

## Skills Demonstrated
- Data Cleaning
- SQL Querying
- Root Cause Analysis
- KPI Reporting
- Dashboard Development
- Forecasting
- Data Visualization

## Initial Questions
- Which maintenance categories create the most downtime?
- Which suppliers create the highest delays?
- What trends impact readiness scores?
- Which bases experience the highest operational risk?

##Tableau Dashboards
1. Executive Summary Dashboard
	- Includes average downtime, total events, readiness score, total maintenance cost
2. Operational Trends Dashboard
	-Includes readiness trends, downtime trends, supplier delays, downtime categories
3. Operational Findings & Recommendations 
	-See Below

##Business Recommendations
Findings:
-Engine Repair & Hydraulics maintenance generated the longest average downtime across all aircraft with both over 18 hours.

-Supplier delays from select vendors were strongly correalted with lower readiness scores and increased maintenance backlog.

-Operational readiness scores declined during periods of elevated maintenance activity, indicating 

Recommendations:
1. Increase preventative maintenance scheduling for high-downtime maintenance categories to reduce operational disruption.
2. Review supplier escalation and inventory management procedures for vendors excedding average delay thresholds.
3. Implement automated KPI monitoring dashboards to identify readiness risks earlier and improve sustainment response times.
4. Prioritize resources allocation during historically high-maintenance periods to stabalize readiness performance
